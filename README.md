Basic Calculator Program
Project Description

This project is a simple calculator program written in Python.
The program allows the user to perform basic mathematical operations using a menu-based system.

The calculator provides several operations such as addition, subtraction, multiplication, division, second power, and square root.
The user can choose an operation from the menu and perform calculations repeatedly until they decide to exit the program.

Features

The calculator supports the following operations:

  Addition

  Subtraction

  Multiplication

  Division

  Second Power (x²)

  Square Root (√x)

  Exit the program

The program will continue running and allow the user to perform calculations multiple times until the Exit option is selected.

How the Program Works

The program displays a calculator menu.

The user selects an operation by entering a number from the menu.

The program asks the user to enter the required numbers.

The calculator performs the selected operation.

The result is displayed.

The menu appears again until the user chooses to exit.
